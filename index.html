Comandos útiles de git

  • git init: comando para iniciar un repositorio de git local (que está en mi computadora)
  • git status: indica el estado de los archivos dentro del repositorio
  • git ad: comando oara agregar archivos al espacio de trabajo
    • Tengo que especificar o el nombre del archivo que quiero agregar, o escribir un "." para indicar que quiero guardar todos los archivos
  • git diff: muestra las diferencias entre los archivos desde la última vez que hice un agregado de cambios con git add y los cambios actuales
  • :Q : comando que me permite salir de la muestra de diferencias y regresar a la carpeta
